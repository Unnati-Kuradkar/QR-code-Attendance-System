# AI-based QR and Face Attendance System

**Author:** Unnati Kuradkar  
**Affiliation:** MCA Student  
**Date:** April 2026  

## Abstract
An AI-powered attendance system combining QR code scanning and facial recognition to automate and secure attendance tracking, reducing manual errors and saving time.  

## Introduction
Manual attendance is slow and error-prone. This project integrates QR codes and facial recognition to ensure fast, accurate, and reliable attendance management.  

## Methodology
Generates unique QR codes for students; facial recognition validates identity. Attendance is recorded in a secure database with timestamps.  

## Implementation
- **Languages:** Python, SQL, PHP, Java  
- **Frameworks/Libraries:** Flask, OpenCV, dlib, Pandas, NumPy, Matplotlib, Seaborn  
- **Tools:** PowerBI, MySQL  

## Results
- Real-time attendance marking with 95%+ facial recognition accuracy.  
- Screenshots:  
  ![Login Page](screenshots/login.png)  
  ![Dashboard](screenshots/dashboard.png)  

## Limitations
- Accuracy drops in low-light conditions.  
- QR codes need printing/distribution.  

## Future Scope
- Mobile app integration  
- Improved lighting/angle robustness  
- Attendance trend analytics  

## Conclusion
A hybrid AI system that automates attendance efficiently using QR codes and facial recognition.
